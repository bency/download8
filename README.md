download8：8comic 下載器
=========


使用說明
=========
適用平台：unix-like

使用方式：

    1. 互動模式

        $ sh download8.sh

    2.命令模式

        $ sh download8.sh --comic-url=http://www.comicbus.com/html/653.html --start-vol=1 --end-vol=2
    

輸入的網址格式如下：http://www.comicbus.com/html/7340.html

如果要下載的漫畫有分集也有分話的 直接輸入第幾集或是第幾話即可
